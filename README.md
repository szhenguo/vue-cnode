## 项目介绍

**cnode官网~使用到技术axios: ^0.18.1 + vue: ^2.6 + vue-router: ^3.0.3 + ant-design-vue: ^1.3.9**

## 项目构建步骤

``` bash
# 安装依赖
$ npm i

# 启动服务 访问 localhost:8888
$ npm start

# 生成环境打包
$ npm run  build

```
